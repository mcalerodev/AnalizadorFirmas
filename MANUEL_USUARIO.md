# Manual de Usuario  
Analizador de Firmas de Archivos

---

## 1. Introducción

El sistema "Analizador de Firmas de Archivos" permite verificar si un archivo es seguro mediante el análisis de su contenido real.

A diferencia de otros sistemas que solo revisan la extensión del archivo, esta herramienta analiza los primeros bytes del archivo para determinar su tipo verdadero y detectar posibles riesgos.

---

## 2. Requisitos

Para utilizar el sistema, se requiere:

- Un navegador web (Google Chrome, Edge u otro)
- Conexión a internet o acceso al servidor donde está alojado el sistema

---

## 3. Acceso al Sistema

1. Abrir el navegador web.
2. Ingresar la dirección del sistema en la barra de direcciones.
3. Esperar a que cargue la página principal.

---

## 4. Pantalla Principal

En la pantalla principal se muestran los siguientes elementos:

- Título del sistema
- Descripción breve del funcionamiento
- Botón para seleccionar archivo
- Área donde se muestra la información del archivo
- Botones de acción

---

## 5. Cómo Subir un Archivo

1. Hacer clic en el botón "Subir archivo".
2. Seleccionar un archivo desde la computadora.
3. Confirmar la selección.

Después de seleccionar el archivo, el sistema mostrará:

- Nombre del archivo
- Tamaño del archivo
- Confirmación de carga exitosa

---

## 6. Cómo Analizar un Archivo

1. Verificar que el archivo haya sido cargado correctamente.
2. Hacer clic en el botón "Analizar".
3. Esperar unos segundos mientras el sistema procesa el archivo.

El sistema realizará automáticamente el análisis del contenido del archivo.

---

## 7. Interpretación de Resultados

Después del análisis, el sistema mostrará:

- Nombre del archivo
- Tipo declarado (extensión del archivo)
- Tipo real detectado

### 7.1 Archivo Seguro

Si el tipo real coincide con la extensión:

- Se mostrará un mensaje indicando que el archivo es válido
- El resultado aparecerá con un indicador visual positivo

Ejemplo:
“El archivo coincide con su extensión y es considerado seguro”

---

### 7.2 Archivo Sospechoso

Si el tipo real no coincide con la extensión:

- Se mostrará una alerta indicando posible riesgo
- El resultado aparecerá con un indicador visual de advertencia

Ejemplo:
“El archivo no coincide con su extensión y puede ser peligroso”

---

## 8. Eliminar Archivo

1. Hacer clic en el botón "Eliminar".
2. Confirmar la acción si el sistema lo solicita.

El archivo será eliminado de la interfaz.

---

## 9. Analizar Otro Archivo

1. Hacer clic en el botón "Analizar otro".
2. Repetir el proceso desde la selección del archivo.

---

## 10. Mensajes del Sistema

El sistema puede mostrar diferentes tipos de mensajes:

### 10.1 Mensaje de éxito
Indica que una acción se realizó correctamente.

Ejemplo:
“Archivo cargado correctamente”

---

### 10.2 Mensaje de error
Indica que ocurrió un problema.

Ejemplo:
“No se pudo cargar el archivo”

---

### 10.3 Mensaje de advertencia
Indica un posible riesgo.

Ejemplo:
“Archivo sospechoso detectado”

---

## 11. Recomendaciones de Uso

- No abrir archivos desconocidos sin analizarlos primero
- Verificar siempre los resultados del análisis
- Evitar subir archivos sospechosos o de origen desconocido

---

## 12. Solución de Problemas

### Problema: El archivo no se carga
Solución:
- Verificar conexión a internet
- Intentar nuevamente

---

### Problema: No se muestra el resultado
Solución:
- Verificar que el archivo fue cargado correctamente
- Hacer clic nuevamente en "Analizar"

---

### Problema: Error inesperado
Solución:
- Recargar la página
- Intentar nuevamente

---

## 13. Conclusión

El sistema proporciona una forma sencilla y efectiva de verificar la autenticidad de archivos, ayudando a prevenir riesgos de seguridad.

Se recomienda utilizar la herramienta antes de abrir archivos desconocidos o descargados de fuentes externas.
